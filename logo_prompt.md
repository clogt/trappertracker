# AI Logo Generation Prompt for TrapperTracker.com

This file contains a detailed prompt and several variations to generate a logo for TrapperTracker.com using an AI image generator (like Midjourney, DALL-E, etc.).

---

### Primary Prompt (Detailed)

Create a modern, vector-style logo for a community safety website called "TrapperTracker". The logo should be clean, easily recognizable, and suitable for a website header and app icon.

**Core Concept:** A stylized map or shield shape that conveys safety and location.

**Key Elements to Include:**
*   A subtle map motif, like contour lines or a simple grid.
*   A single, prominent red marker icon (like a warning "blip" or a sharp-angled pin) to represent a danger zone.
*   A contrasting, friendly-colored (blue or green) map pin icon that contains a small paw print, representing a lost or found pet.
*   The name "TrapperTracker" integrated cleanly, either below or alongside the icon. Use a modern, sans-serif typeface.

**Color Palette:**
*   Primary: Deep Red (#D92E2E), Safety Blue (#3498DB), and a neutral Dark Gray/Charcoal (#333333).
*   Background: White or off-white.

**Style:** Flat design, minimalist, high-contrast, with clean lines and no gradients. The overall feeling should be professional and trustworthy, yet compassionate.

**Example Prompt:**
> A modern vector logo for 'TrapperTracker', designed in a flat, minimalist style. The logo features a stylized shield outline. Inside the shield, there's a simple map grid. On the map, place a sharp, red warning marker and a separate, friendly blue map pin with a paw print inside it. The name 'TrapperTracker' is written in a clean, sans-serif font below the shield. High contrast, no gradients, on a white background.

### Tips for Refining the Logo

*   **Simplify:** If the AI generates something too complex, add words like "simplified," "single icon," or "two elements only."
*   **Change Colors:** Experiment with different color palettes by specifying them, e.g., "Use a color palette of orange, teal, and dark grey."
*   **Font:** If you don't like the text, you can ask the AI to generate the "icon only" and add the text later using graphic design software for better control.
*   **Format:** Add "app icon" or "website favicon" to see how the logo looks in a small, square format.
